 ==== Joc d'Escacs en Java ====

==== Descripció ====

Aquest codi implementa un joc d'escacs per consola creat amb java.
Permet a dos jugadors realitzar moviments, els quals es validen amb les normes de cada peça.

==== La tasca ha servit per treballar: ====

- Arrays de 2 dimensions (Programació)
- Clean Code (Entorns)
- Diagrames UML (Entorns)
- Proves automatitzades amb JUnit (Entorns)
  
==== Execució del programa ==== 

1. Clona o descarrega el repositori
2. Obre el projecte amb Visual Studio Code
3. Executa la classe: Escacs.java
